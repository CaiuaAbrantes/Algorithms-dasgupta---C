# Algorithms-dasgupta---C

Este repositório reúne códigos desenvolvidos por mim durante a minha graduação, com o objetivo exclusivamente acadêmico e voltado ao estudo de algoritmos e estruturas de dados. As implementações aqui presentes foram criadas como parte do meu processo de aprendizado, seja em atividades de disciplinas, exercícios propostos em sala de aula ou estudos individuais para melhor compreensão dos conceitos abordados ao longo do curso.

Os códigos refletem experimentações práticas com diferentes técnicas e algoritmos clássicos, servindo como registro da evolução do meu raciocínio e da forma como os problemas foram abordados naquele momento. Não há a intenção de apresentar soluções definitivas, otimizadas ou prontas para uso em ambientes de produção, mas sim de documentar o aprendizado e a prática adquiridos durante a formação acadêmica.

Assim, este repositório deve ser entendido como um material de apoio e consulta, representando uma etapa do meu desenvolvimento como estudante, e podendo futuramente ser revisado, aprimorado ou utilizado como base para estudos mais aprofundados.
